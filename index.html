常见问题

FAQ

快速找到您想要的答案

TechTutorial Pro 是一个纯静态网站，支持所有现代浏览器，包括 Chrome、Edge、Safari、Firefox。同时完美适配 Windows、macOS、Linux、iOS 和 Android 系统。得益于 PWA 技术，您甚至可以将网站添加到手机桌面，获得接近原生应用的体验。

访问我们的教程中心，选择您感兴趣的分类开始学习。教程支持 Markdown 格式，包含代码高亮、复制按钮、步骤导航和目录导航。支持图片、GIF 和视频嵌入，提供最佳的学习体验。

完全免费！TechTutorial Pro 基于 GitHub Pages 部署，永久免费，无广告。所有教程、软件下载、固件更新和源代码均免费提供。我们致力于为全球开发者提供高质量的技术教育资源。

支持！TechTutorial Pro 使用 Service Worker 技术实现了离线缓存。首次访问后，核心内容和已浏览的教程页面将自动缓存到本地。即使没有网络连接，您也可以正常浏览已缓存的教程内容。您还可以通过 PWA 功能将网站安装到桌面。

我们欢迎社区贡献！请查看我们的贡献指南了解详情。您可以通过 GitHub Fork 本项目，添加或修改教程内容后提交 Pull Request。教程内容位于 tutorials/data/ 目录下的 JSON 文件中。

非常简单！只需 Fork 本项目到您的 GitHub 账号，GitHub Actions 将自动部署到 GitHub Pages。无需任何额外配置。您只需修改 tutorials/data/ 中的教程内容、替换 assets/img/ 中的图片，然后 Push 到 main 分支即可自动更新网站。
